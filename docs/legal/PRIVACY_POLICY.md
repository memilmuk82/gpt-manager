# 개인정보처리방침

종로산업정보학교는 GPT Share Manager vNext, 표시명 생성형 AI 계정 공동 사용 지원 시스템을 운영하면서 개인정보 보호 관련 법령과 학교 내부 지침을 준수하기 위해 다음과 같이 개인정보처리방침을 수립합니다.

본 방침은 법적 검토 전 운영 안내 문서입니다. 학교 또는 기관의 공식 개인정보처리방침, 개인정보보호 담당자 검토 결과, 관련 법령 해석과 충돌하는 경우 공식 지침과 최종 검토 결과를 우선합니다.

## 1. 개인정보 처리 목적

종로산업정보학교는 다음 목적을 위해 개인정보를 처리합니다.

### 로그인 및 사용자 식별

- 로컬 로그인 또는 Google OAuth 로그인을 통한 사용자 식별
- 승인 사용자, 승인 대기 사용자, 관리자, 보조관리자 구분
- 정지 계정 접근 제한

### 예약 관리

- 공용 생성형 AI 계정 사용 예약 등록
- 예약 충돌 확인
- 오늘 예약, 내 예약, 월간 예약 조회
- 예약 취소 및 완료 처리
- 예약 시점의 사용 규칙 동의 버전 기록

### 사용 기록 관리

- 사용 목적, 작업 유형, 요약, 프롬프트, 결과 메모 기록
- 사용자별 사용 이력 확인
- 운영 투명성 및 사후 점검 지원

### 프롬프트 정리

- 사용자가 입력한 요청의 구조화와 프롬프트 정리
- OpenAI, Google Gemini, Anthropic Claude API를 활용한 개선 의견 생성
- 정리 결과 저장 및 재조회
- 사용자별 최근 24시간 호출 제한 관리

### 관리자 기능

- 사용자 승인 상태, 활성 여부, 권한, 인증번호 담당자 여부 관리
- 사용자 목록과 운영 현황 확인
- 예약, 사용 로그, 프롬프트 정리 수 요약 확인
- 월간 운영 보고서, 감사 로그, CSV 내보내기, DB 백업 등 운영 관리

## 2. 처리하는 개인정보 항목

### 로컬 로그인 및 사용자 계정

로컬 계정을 사용하는 경우 다음 항목을 처리합니다.

- 아이디로 사용하는 이메일 주소
- 이름
- 부서
- 내선번호
- 비밀번호 해시
- 인증 제공 방식
- 승인 상태
- 권한 정보
- 계정 활성 여부
- 인증번호 담당자 여부
- 정렬 순서
- 계정 생성 및 수정 시간

Google OAuth 로그인을 사용하는 경우 다음 항목을 처리할 수 있습니다.

- 이메일 주소
- 표시 이름
- 이메일 검증 여부

- 승인 상태와 권한 정보

### 예약 정보

서비스 이용 과정에서 다음 예약 정보가 저장될 수 있습니다.

- 예약 사용자
- 사용할 생성형 AI 리소스
- 시작 시간과 종료 시간
- 작업 유형
- 사용 목적
- 작업 설명
- 안전 확인 여부
- 사용 규칙 동의 버전
- 예약 상태
- 생성 및 수정 시간

### 사용 로그와 프롬프트 정리 정보

서비스 이용 과정에서 다음 정보가 저장될 수 있습니다.

- 사용 로그 작성자
- 연결된 예약과 리소스
- 작업 유형
- 사용 요약
- 프롬프트 내용
- 결과 메모
- 프롬프트 정리 목표

- 프롬프트 정리 결과
- 사용 모델명
- 생성 및 수정 시간

### LLM API Key

사용자가 프롬프트 정리 기능을 사용하기 위해 LLM API Key를 등록하는 경우 다음 항목을 처리합니다.

- 서버에서 암호화된 API Key ciphertext
- API Key 마지막 4자리
- 제공자 이름
- 생성 및 수정 시간

API Key 원문은 데이터베이스에 저장하지 않으며, 화면에 다시 표시하지 않습니다.

### 관리자 작업 및 감사 로그

관리자 기능 사용 과정에서 다음 정보가 저장될 수 있습니다.

- 작업 수행 관리자
- 작업 종류
- 작업 대상 유형과 식별자
- 작업 요약

- 공용 생성형 AI 계정 ID/PW

## 4. 개인정보 보유 및 이용 기간

개인정보와 서비스 이용 기록은 시스템 운영, 사용자 식별, 예약 관리, 사용 기록 확인, 보안 사고 대응, 운영 감사 목적에 필요한 기간 동안 보관합니다.

운영 목적이 종료되거나 사용자가 삭제를 요청하고 운영상 보관 필요성이 없다고 판단되는 경우 관련 기록을 삭제하거나 비식별 처리할 수 있습니다.

다만 학교 내부 감사, 보안 사고 대응, 서비스 운영 확인, 법령상 보관 필요 등 합리적인 사유가 있는 경우 필요한 범위에서 일정 기간 보관할 수 있습니다.

SQLite 백업 파일에는 삭제 전 정보가 일정 기간 남을 수 있으므로, 운영자는 백업 보관 기간과 폐기 절차를 별도로 관리해야 합니다. 구체적인 보관 기간은 학교 또는 기관의 개인정보보호 지침에 따라 최종 확정합니다.

## 5. 개인정보의 파기 절차 및 방법

처리 목적이 달성되었거나 보관 필요성이 없어진 개인정보는 운영자가 확인 후 삭제합니다.

전자적 파일 형태의 정보는 복구하기 어렵도록 삭제하고, 백업 파일은 정해진 보관 기간이 지나면 삭제합니다. 다만 시스템 무결성, 감사, 보안 사고 대응을 위해 일부 로그는 필요한 범위에서 일정 기간 보관될 수 있습니다.

## 6. 외부 서비스 이용

### Google OAuth

Google OAuth 로그인 과정에서 Google 계정 인증 화면과 Google OAuth API가 사용됩니다. Google OAuth의 인증 절차, 계정 정보 처리, 보안 정책은 Google의 정책을 따릅니다.

### OpenAI, Google Gemini, Anthropic Claude API

프롬프트 정리 기능을 사용하는 경우 사용자가 입력한 프롬프트, 정리 목표, 정리를 위해 조립된 요청 내용이 OpenAI, Google Gemini, Anthropic Claude API 처리 목적으로 해당 외부 Provider에 전송될 수 있습니다.

이용자는 OpenAI, Google Gemini, Anthropic Claude API로 전송되는 프롬프트에 학생 개인정보, 학생 성적, 상담 내용, 생활기록부 민감정보, 평가 문항 원본, 비공개 내부 자료를 입력해서는 안 됩니다.

OpenAI, Google Gemini, Anthropic Claude API의 데이터 처리, 보안, 보관 정책은 Google의 API 및 서비스 정책을 따릅니다.

## 7. 개인정보의 제3자 제공 및 외부 전송

종로산업정보학교는 법령에 근거가 있거나 정보주체의 동의가 있는 경우를 제외하고 개인정보를 제3자에게 제공하지 않습니다.

다만 Google OAuth와 OpenAI, Google Gemini, Anthropic Claude API 사용 과정에서 인증 또는 API 처리 목적상 필요한 정보가 외부 서비스 제공자에게 전송될 수 있습니다. 외부 서비스 제공자, 전송 항목, 이용 목적, 보관 정책 등 세부 사항은 해당 서비스의 정책과 학교 또는 기관의 개인정보보호 지침에 따라 최종 확인해야 합니다.

## 8. 개인정보 처리 위탁

현재 본 시스템 문서에는 별도의 개인정보 처리 위탁 업체를 두는 구조가 명시되어 있지 않습니다.

다만 서버 운영, 도메인, 클라우드, 외부 API, 유지보수 과정에서 위탁 또는 외부 서비스 이용에 해당하는 관계가 발생할 수 있으므로, 실제 운영 전 기관 개인정보보호 담당자가 위탁 여부와 고지 필요성을 확인해야 합니다.

## 9. 정보주체의 권리와 행사 방법

이용자는 본인 개인정보에 대해 열람, 정정, 삭제, 처리정지를 요청할 수 있습니다.

요청은 정보관리책임자에게 이메일, 전화, 내부 업무 절차를 통해 할 수 있습니다. 운영자는 본인 확인 후 관련 법령과 학교 내부 지침에 따라 처리합니다.

다만 다른 이용자의 권리 침해, 운영 감사, 보안 사고 대응, 법령상 보관 필요 등 정당한 사유가 있는 경우 일부 요청이 제한될 수 있습니다.

## 10. 안전성 확보 조치

본 시스템은 개인정보와 인증 정보를 보호하기 위해 다음 조치를 적용합니다.

- 비밀번호 해시 저장
- 선택 시 LLM API Key 서버 암호화 저장
- 프론트엔드에 API Key 전체 값 노출 금지
- HTTPS 기반 운영
- 세션 쿠키 HttpOnly 설정
- SameSite 쿠키 설정
- 운영 환경 SECRET_KEY 사용
- 승인 상태와 권한에 따른 접근 제한
- CSRF 토큰 기반 POST 요청 보호
- 사용자 입력 HTML 실행 방지
- 관리자 작업 감사 로그 기록
- SQLite 백업/복원 절차 문서화

## 11. 이용자의 주의사항

이용자는 다음 사항을 준수해야 합니다.

- 본인 계정만 사용합니다.
- 비밀번호와 인증 정보를 타인에게 공유하지 않습니다.

- 외부 AI 서비스의 약관과 학교 내부 지침을 준수합니다.

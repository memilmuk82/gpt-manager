# 이용약관

## 제1조 목적

이 약관은 종로산업정보학교가 운영하는 GPT Share Manager vNext, 표시명 생성형 AI 계정 공동 사용 지원 시스템의 이용 조건과 절차, 이용자와 운영자의 권리와 의무, 책임 사항을 정하는 것을 목적으로 합니다.

본 시스템은 학교 공용 생성형 AI 계정의 사용 예약, 사용 기록 관리, 안전한 활용 안내, Gemini 기반 프롬프트 점검을 지원하기 위한 내부 업무 지원 서비스입니다.

## 제2조 서비스 소개

GPT Share Manager vNext는 공용 생성형 AI 계정을 직접 제어하거나 외부 AI 서비스의 계정 로그인 상태를 통제하는 서비스가 아닙니다.

본 시스템은 다음 기능을 제공합니다.

- 공용 AI 계정 예약 신청 및 현황 조회
- 오늘 예약 및 내 예약 확인
- 사용 목적과 사용 기록 관리
- 프롬프트 및 결과 메모 기록
- 사용자별 Gemini API Key 암호화 저장
- Gemini API를 활용한 프롬프트 점검 및 개선 의견 제공
- 관리자 및 보조관리자의 사용자 승인, 정지, 운영 현황 확인
- 생성형 AI 계정 사용 안내와 보안 주의사항 제공

## 제3조 이용 대상과 계정

본 시스템은 종로산업정보학교의 승인된 사용자 또는 관리자가 승인한 사용자에게 제공됩니다.

사용자는 로컬 계정 또는 Google OAuth 계정으로 로그인할 수 있습니다. 학교 도메인 계정은 설정에 따라 자동 승인될 수 있으며, 그 외 계정은 관리자 승인 후 사용할 수 있습니다.

관리자와 보조관리자는 사용자 승인, 정지, 예약 및 사용 현황 확인 등 운영 관리 기능을 수행할 수 있습니다.

## 제4조 공용 AI 계정 예약 서비스

사용자는 공용 생성형 AI 계정 사용 전에 예약을 등록해야 합니다.

예약 등록 시 다음 사항을 입력하거나 선택합니다.

- 사용할 생성형 AI 계정 리소스
- 시작 시간과 종료 시간
- 사용 목적

같은 리소스에 대해 시간이 겹치는 예약은 등록할 수 없습니다. 사용자는 예약 시간과 사용 목적을 사실에 맞게 입력해야 하며, 사용 후 필요한 경우 예약 완료 또는 취소 상태를 정리해야 합니다.

## 제5조 사용 기록 관리

사용자는 공용 생성형 AI 계정 사용 후 작업 유형, 요약, 프롬프트, 결과 메모 등을 기록할 수 있습니다.

사용 기록은 운영 투명성, 사용 현황 확인, 추후 업무 개선을 위해 사용됩니다. 사용자는 실제 업무 내용을 과도하게 상세히 입력하지 않아야 하며, 개인정보나 민감정보가 포함되지 않도록 주의해야 합니다.

## 제6조 Gemini 기반 프롬프트 점검 기능

본 시스템은 사용자가 입력한 프롬프트를 Gemini API로 전송하여 명확성, 안전성, 수업 활용성 등을 점검하고 개선 의견을 제공합니다.

Gemini 기반 프롬프트 점검 결과는 참고용입니다. AI가 생성한 결과의 정확성, 적법성, 교육적 적합성은 최종적으로 이용자가 검토해야 합니다.

Gemini API 사용 시 프롬프트 내용이 API 처리 목적으로 외부 서비스에 전송될 수 있습니다. 학생 개인정보, 평가 자료, 생활기록부 민감정보, 상담 내용, 비공개 업무자료는 입력하지 않아야 합니다.

## 제7조 이용자의 의무

이용자는 다음 행위를 해서는 안 됩니다.

- 타인의 계정으로 로그인하거나 타인의 계정을 무단 사용하는 행위
- 허위 정보로 예약하거나 사용 기록을 작성하는 행위
- 공용 생성형 AI 계정의 접속 정보를 무단 공유하는 행위
- 서비스 운영을 방해하거나 시스템에 무리한 요청을 반복하는 행위
- 개인정보, 학생 민감정보, 평가 보안 자료를 입력하는 행위
- 법령, 학교 규정, 외부 AI 서비스 정책을 위반하는 행위

## 제8조 개인정보 및 학생 민감정보 입력 금지

이용자는 본 시스템과 외부 AI 서비스에 다음 정보를 입력해서는 안 됩니다.

- 주민등록번호, 주소, 연락처, 계좌번호 등 고유식별정보와 개인정보
- 학생 성적, 상담 내용, 생활기록부 민감정보
- 평가 문항 원본, 정답지, 채점 기준, 시험은행 문항
- 비공개 내부 문서, 인사 자료, 징계 자료, 급여 자료
- 그 밖에 공개되어서는 안 되는 학교, 학생, 교직원 관련 정보

## 제9조 외부 AI 서비스 정책

본 시스템은 OpenAI, Google Gemini 등 외부 AI 서비스를 직접 운영하지 않습니다.

외부 AI 서비스의 계정, 인증, 사용 제한, 데이터 처리, 결과물 정책은 각 서비스 제공자의 약관, 개인정보처리방침, API 정책을 따릅니다.

이용자는 외부 AI 서비스를 사용할 때 해당 서비스의 정책과 학교 내부 지침을 함께 준수해야 합니다.

## 제10조 AI 결과에 대한 면책

AI가 제공하는 답변, 분석, 요약, 개선안은 참고 자료입니다.

종로산업정보학교와 시스템 운영자는 AI 결과의 정확성, 완전성, 최신성, 특정 목적 적합성을 보장하지 않습니다. 이용자는 AI 결과를 수업, 행정, 평가, 학생 지도 등에 활용하기 전에 직접 검토하고 필요한 경우 관련 규정과 책임자 확인을 거쳐야 합니다.

AI 결과를 무비판적으로 사용하여 발생한 손해, 오류, 분쟁에 대해서는 이용자가 최종 책임을 집니다.

## 제11조 서비스 제한 및 중단

운영자는 다음 경우 서비스 이용을 제한하거나 일시 중단할 수 있습니다.

- 시스템 점검, 장애 복구, 보안 조치가 필요한 경우
- 사용자가 약관 또는 보안 지침을 위반한 경우
- 외부 AI 서비스 장애 또는 정책 변경으로 기능 제공이 어려운 경우
- 학교 운영상 서비스 제공이 어렵다고 판단되는 경우

# 법률 검토 체크리스트

이 문서는 서비스 운영 전 이용약관과 개인정보처리방침을 법률 전문가 또는 기관 개인정보보호 담당자가 검토할 때 확인할 항목입니다. 법률 자문 자체를 대체하지 않습니다.

## 이용약관

```text
서비스 목적과 실제 기능 범위가 일치하는가
공용 생성형 AI 계정 ID/PW를 저장하지 않는다는 설명이 명확한가
사용자가 입력하면 안 되는 개인정보/민감정보/평가 보안 자료가 명확한가
예약/사용 로그/프롬프트 점검 결과의 보관 목적과 책임 범위가 명확한가
서비스 중단, 계정 정지, 관리자 권한 행사 기준이 명확한가
```

## 개인정보처리방침

```text
수집 항목: 이메일, 이름, 부서, 내선번호, 예약/사용 로그, API Key 암호문 등
수집 목적: 사용자 인증, 예약 관리, 사용 이력 관리, 프롬프트 점검 기능 제공
보관 기간과 삭제 절차
관리자/보조관리자의 접근 범위
Google OAuth 사용 시 외부 인증 제공자 정보 처리 설명
Gemini API 호출 시 사용자의 개인 API Key 사용 구조 설명
```

## 운영 전 확인

```text
ENABLE_REVIEW_ADMIN=false 확인
REVIEW_ADMIN_PASSWORD 운영 재사용 금지
ALLOWED_GOOGLE_DOMAIN 운영 도메인 정책 확인
SESSION_COOKIE_SECURE=true 확인
WTF_CSRF_ENABLED=true 확인
SQLite 백업/복원 리허설 완료
```

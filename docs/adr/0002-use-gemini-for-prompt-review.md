# ADR 0002 - Gemini API를 프롬프트 점검기에 사용

## 결정

Gemini API는 자유 챗봇이 아니라 프롬프트 점검/개선 기능에 사용한다.

## 이유

```text
연수 요구사항의 API 활용을 충족한다.
경량 모델로도 충분한 작업이다.
비용 통제가 쉽다.
교사가 프롬프트 엔지니어링 개념을 몰라도 구조화된 프롬프트를 배울 수 있다.
```

## 제외

```text
자유 채팅
수업안/평가계획 생성
교육과정 파일 분석
학생정보 기반 생성
```

## 결과

앱은 AI 생성기가 아니라 AI 사용 운영·개선 도구가 된다.

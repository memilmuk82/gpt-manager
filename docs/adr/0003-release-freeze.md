# ADR 0003 - 2026-07-02 RC1 및 2026-07-03 Release Freeze

## 결정

2026-07-02에 RC1을 만들고, 2026-07-03에는 기능 추가와 리팩토링을 하지 않는다.

## 이유

```text
제출 당일 코드 수정은 위험하다.
OAuth, Docker, OCI, API 호출은 마지막 날에 삽질할 가능성이 크다.
완성도보다 안정적인 시연이 중요하다.
```

## 허용 작업

```text
최종 테스트
README 오타 수정
시연 설명 정리
스크린샷 추가
```

## 금지 작업

```text
신규 기능
대규모 리팩토링
DB 구조 변경
인증 정책 변경
Gemini 호출 구조 변경
```

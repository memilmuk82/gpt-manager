BYOK 보안 안내

이 기능은 사용자가 본인의 API Key를 등록하여 사용하는 방식입니다. API 사용 요금은 각 Provider 정책에 따라 사용자 본인에게 청구될 수 있습니다.

키 발급, 결제 설정, 사용량 관리는 사용자 본인의 책임입니다. 테스트 호출도 소량의 비용이 발생할 수 있습니다.

API Key는 기본적으로 서버 DB에 저장하지 않습니다. 저장 옵션을 선택한 경우에도 평문이 아닌 암호화된 형태로 저장됩니다.

관리자와 다른 사용자는 API Key 원문을 볼 수 없습니다. 공용 PC에서는 API Key 저장 옵션을 사용하지 마세요.

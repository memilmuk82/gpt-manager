BYOK Security

BYOK 보안 안내

관리자는 공용 API Key를 제공하지 않으며, 다른 사용자의 키를 대신 등록하거나 조회하지 않습니다.

API 사용 요금은 각 Provider 정책에 따라 사용자 본인에게 청구될 수 있습니다. 연결 테스트, 모델 목록 새로고침, 프롬프트 정리 실행도 비용이나 사용량 차감이 발생할 수 있습니다.

API Key는 기본적으로 서버 DB에 저장하지 않습니다. 저장 옵션을 선택한 경우에도 `LLM_KEY_ENCRYPTION_SECRET` 기반 암호화 값과 마지막 4자리만 저장합니다.

관리자와 다른 사용자는 API Key 원문을 볼 수 없습니다. 공용 PC나 공동 계정 환경에서는 서버 저장과 브라우저 저장 옵션을 사용하지 마세요.

학생 개인정보, 성적, 생활기록부 원문, 상담 내용, 평가 문항 원본, 비공개 내부 자료는 입력하지 마세요.
